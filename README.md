# Understand Java spring boot API

## 1 \ pom.xml (its compared to package.json in node app)

- pom.xml stands for Project Object Model and belongs to Apache Maven.
- The Most Important File
- This is the heart of every project configuration
- It defines:
  - Dependencies (libraries your app uses)
  - Java version
  - Plugins
  - Build settings
  - Project metadata
